# Smart Hostel Room Allocation System

## Overview
This project is a web application that manages hostel rooms and automatically allocates rooms to students based on capacity and facility requirements.

## Features
- Add new hostel rooms
- View all rooms
- Search rooms based on requirements
- Automatically allocate smallest suitable room

## Tech Stack
- React
- Vite
- JavaScript
- CSS

## Run Locally
npm install
npm run dev
